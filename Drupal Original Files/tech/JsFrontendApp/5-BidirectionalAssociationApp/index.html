A Plain JavaScript Web App with Bidirectional Associations

  • Authors
  • Publishers
  • Books

This app, which is accompanied by a tutorial, is the 5th example app of our incremental sequence of six examples of Plain JS Front-End Web Apps.

This app supports the following operations:

  • Manage author data
  • Manage publisher data
  • Manage book data

For playing with the app, you may first want to and then view it (e.g. by choosing "Manage book data" and then "List all books"). For observing the internal status and error messages, turn on the JavaScript console.

This JavaScript front-end data management app takes care of the object types Author, Publisher and Book as well as the two bidirectional associations that assign a publisher and (one or more) authors to a book and, vice versa, zero or more books to an author and to a publisher. The app supports the four standard data management operations (Create/Retrieve/Update/Delete). It extends the Unidirectional Association App by adding code for handling derived inverse reference properties.
